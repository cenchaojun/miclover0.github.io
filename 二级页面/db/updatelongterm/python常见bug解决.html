主要存放python使用中常见bug的修复

Bug与Solution

    一、Assign requires shapes of both tensors to match. lhs shape= [512,5721] rhs shape= [512,169]

      原因是自己直接拷贝之前跑过的代码的时候，顺带着连上次的日志也带过来了，所以解决办法直接把日志文件删掉，再次运行就OK了。


    二、UnicodeDecodeError: 'ascii' codec can't decode byte 0xe6 in position 36: ordinal not in range(128)

      在需要的地方添加如下代码

    import sys

    reload(sys)

    sys.setdefaultencoding('utf8')
