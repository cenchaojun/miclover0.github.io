###本地与github的连接<br>

1、下载git软件，安装；<br>
2、在本地新建文件，作为本地文件库；<br>
3、打开git bush  ->  git clone https://miclover0.github.io/id/ <br>
4、线下修改代码<br>
5、接下来输入：<br>
&nbsp;git add . <br>
&nbsp;git config --global user.email "miclover0@163.com"  <br>
&nbsp;git config --global user.name "miclover0"<br>
&nbsp;git commit -m tag<br>
&nbsp;git push<br>
